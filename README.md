# 🌊 Flood Prediction Using Satellite Images

This project focuses on building a **Flood Prediction System** using **satellite imagery** and machine learning. The system leverages deep learning models to detect flood-prone areas and predict flooding events by analyzing remote sensing data.

---

## 🚀 Project Overview
Floods are one of the most devastating natural disasters, affecting millions worldwide. Early prediction can minimize damage and save lives.  
This project aims to:
- Use **satellite images** to identify patterns in flood-prone regions.  
- Train a machine learning model to predict the likelihood of flooding.  
- Provide insights for disaster management and early warning systems.  

---

## 📂 Dataset
- **Source:** Satellite imagery dataset (Sentinel-1/2, Landsat, or other flood datasets).  
- **Preprocessing:**  
  - Image cleaning and normalization  
  - Segmentation of flooded vs non-flooded areas  
  - Labeling based on ground truth data  

---

## 🧠 Methodology
1. **Data Preprocessing**  
   - Image resizing and normalization  
   - Splitting into train/test sets  

2. **Model Development**  
   - CNN (Convolutional Neural Network) for feature extraction  
   - Classification into *flooded* vs *non-flooded* categories  

3. **Evaluation**  
   - Accuracy, Precision, Recall, F1-score  
   - Confusion Matrix for model validation
